## How to Update This Changelog

1. **Add entries under [Unreleased]** as you make changes
2. **Use the correct category**:
   - `Added` for new features
   - `Changed` for changes in existing functionality
   - `Deprecated` for soon-to-be removed features
   - `Removed` for now removed features
   - `Fixed` for any bug fixes
   - `Security` for vulnerability fixes
3. **When releasing**, move Unreleased items to a new version section
4. **Include dates** in YYYY-MM-DD format
5. **Link to commits/PRs** when helpful
